# Tailor Web App 👗🧵

A simple web-based application for tailors and customers to manage tailoring services, registrations, and logins.

## 📌 Features

- Customer Login Page
- Tailor Login Page
- Tailor Registration Page
- Home Page
- Screenshots of the UI
